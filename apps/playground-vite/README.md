# Compile TypeScript to JavaScript Vite plugin

This module huntersofbook team created.

## Features

- Zero-config required
- Hot reload
- Compile TypeScript to JavaScript
- Support TypeScript config

## Requirements
- Node.js >= 16.0.0
- Vite >= 3.1.8

- Clone this repository
